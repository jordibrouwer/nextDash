Quick Start (30 sec)

  • Open search with >, type a bookmark, press Enter.
  • Open recent bookmarks with *.
  • Open keyboard cheatsheet with ! or Ctrl+/.
  • Open config and tune dashboard layout, sorting, and smart collections.

Navigation

  • Switch pages with number keys 1-9.
  • Move between pages with Shift + Left/Right.
  • Navigate bookmarks with Up/Down and Left/Right.
  • Open selected bookmark with Enter or Space.
  • Inline edit selected bookmark with E; cancel with Esc.

Search & Input Modes

  • > open normal search, : commands, ? finders.
  • / fuzzy search behavior depends on the configured search mode.
  • Use filters in search: category:work, status:online, page:2.
  • Turn on "Keep search open when empty" for fast repeated searches.

Smart Collections

  • Recently opened, Stale bookmarks, and Most used are generated automatically.
  • Each smart collection can be enabled per page and limited in visible items.
  • Set limit to 0 for unlimited items.
  • Smart sections support collapse/expand on dashboard.

Sorting, Pinned & Categories

  • Pinned bookmarks always stay on top of non-pinned bookmarks.
  • Pinned state is shown with a compact pin icon on dashboard rows.
  • Pinned bookmarks are always sorted alphabetically.
  • Non-pinned bookmarks use your selected sort mode (manual, A-Z, recent, pinned-first behavior).
  • Categories can be collapsed and can default to collapsed.

Manage Bookmarks

  • Add, edit, reorder, move, and remove bookmarks in config.
  • Bulk actions: select, move category, toggle pin, delete.
  • On dashboard, reorder by long-pressing a bookmark and then dragging.
  • Use Alt+Up / Alt+Down to reorder quickly in config lists.
  • Conflict highlighting warns about duplicate URLs and shortcuts before save.

Finders

  • Format: ?shortcut your query (example: ?g nextdash).
  • Use %s in finder URL where query should be inserted.
  • If bookmark and finder share a shortcut, both options are shown in search context.

Status Monitor

  • Enable status checks to display online/offline state.
  • Optionally show ping and loading indicators.
  • Use "Skip fast ping check" to reduce noise and speed up checks.

Tips Rotation Above Buttons

  • Tips rotate every 5 to 8 seconds.
  • After every 5 normal tips, a priority tip is shown.
  • Priority tips include recent, cheatsheet, navigation, and quick-add shortcuts.
  • You can toggle tips in config#general under Dashboard.

Backup, CSV & Import

  • ZIP backup exports pages, bookmarks, categories, settings, and themes.
  • Importing ZIP replaces current data.
  • CSV export/import focuses on bookmarks.
  • Best practice: create a backup before large edits or imports.

Theme, Layout & Branding

  • Choose theme, font size, font weight, and layout preset.
  • Enable auto dark mode to follow system appearance.
  • Upload custom favicon and custom font.

Data, Storage & Device-Specific Settings

  • Server-side data is stored in /data.
  • Device-specific settings are stored in browser local storage.
  • When device-specific mode is enabled, visual behavior can differ between devices.

Troubleshooting

  • No bookmarks visible: check page selector and category filter in config.
  • Sorting seems wrong: verify sort mode and pin state.
  • Tips not visible: enable "Show tips above buttons" in Dashboard settings.
  • Search misses items: try fuzzy search and filter syntax.
